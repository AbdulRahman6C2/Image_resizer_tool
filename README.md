# 🖼️ Image Resizer Tool

## 📌 Description
This script resizes all images in the `images` folder to 300x300 pixels and converts them to PNG format. Resized images are saved in the `resized_images` folder.

## 🛠️ Technologies Used
- Python
- Pillow (PIL)
